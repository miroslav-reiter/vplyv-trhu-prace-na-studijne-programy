# Prvky motivačného modelu

## Hodnoty

| ArchiMate prvok | Názov prvku |
|---|---|
| Value | Relevancia študijných programov pre potreby trhu práce |
| Value | Vyššia uplatniteľnosť absolventov vysokých škôl |
| Value | Plynulý prechod absolventov do pracovného prostredia |
| Value | Strategické riadenie programovej ponuky vysokých škôl |
| Value | Efektívne využívanie verejných zdrojov vo vysokoškolskom vzdelávaní |
| Value | Rozvoj perspektívnych profesijných oblastí |
| Value | Kvalita a aktuálnosť obsahu študijných programov |

| ArchiMate prvok | Názov prvku |

| Driver | Nesúlad medzi ponukou študijných programov a potrebami praxe |

| Driver | Nedostatočné využívanie dát pri riadení študijných programov |
| Driver | Slabé prepojenie vysokých škôl so zamestnávateľmi |
